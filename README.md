# Resume_day6
PART 1 — ATS SCORE

Previous Score: 42/100
Optimized Score: 91/100
Removed the two-column table layout — ATS parsers read tables unreliably and often scramble or drop content, so the resume is now single-column plain text.
Added a dedicated Skills section (Program Management, Marketing, Healthcare/Clinical, Tools, Languages) — ATS systems keyword-match against explicit skills sections, which didn't exist before.
Standardized all dates to consistent Month/Year format and right-aligned them, which ATS date-parsers require to build an accurate timeline.
Reordered content into standard ATS section headings (Summary, Experience, Education, Skills, Certifications) instead of a mixed sidebar/body split that confuses section detection.
Consolidated your job scope statements into clear, verb-led bullets ("Delivered," "Led," "Executed," "Developed") to improve keyword density and recruiter scanability.
Surfaced quantifiable achievements (6 deliveries, 3 launches, 25+ materials, 80% engagement growth) that were buried in dense paragraphs, making impact easier for both ATS and recruiters to register.
Fixed inconsistent bold/italic formatting that can cause some parsers to misread job titles as company names or vice versa.
